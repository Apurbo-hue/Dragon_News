# Dragon News Portal

**Live Demo (Firebase):** [(https://dragon-news-e05dc.firebaseapp.com/)]  
**Live Demo (Netlify):** [(https://dragon-news-apurba.netlify.app/)]  

---

## Description

This is a **news portal** where news articles are displayed according to their categories. The portal includes a **login system** and **register system** for users.  

> ⚠️ Note: Google and GitHub login functionality has **not been implemented yet**.

---

## Features

- Category-wise news display  
- User authentication (email/password)  
- Responsive design for mobile and desktop  

---

## Tech Stack

- **Frontend:** React, Tailwind CSS, Vite  
- **Backend / Hosting:** Firebase Hosting / Netlify  
- **Authentication:** Firebase Authentication (Email/Password)
